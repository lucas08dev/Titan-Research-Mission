# 🚀 Titan Research Mission

Sistema Inteligente de Monitoramento Espacial desenvolvido em Python para simulação e análise operacional de uma missão espacial experimental.

---

# 📌 Sobre o Projeto

O projeto **Titan Research Mission** foi desenvolvido com o objetivo de criar um sistema capaz de monitorar condições operacionais de uma missão espacial.

O sistema recebe dados simulados da operação, interpreta as informações e executa análises automáticas para identificar possíveis situações críticas durante a missão.

O projeto foi desenvolvido utilizando conceitos de:

- lógica de programação;
- algoritmos;
- pensamento computacional;
- automação básica;
- inteligência artificial introdutória;
- sustentabilidade energética.

---

# 🎯 Objetivo

Desenvolver um sistema inteligente capaz de:

✅ Receber dados simulados da missão espacial  
✅ Interpretar condições operacionais  
✅ Gerar alertas automáticos  
✅ Executar respostas automatizadas  
✅ Organizar informações da missão  
✅ Simular monitoramento espacial em tempo real  

---

# 🛰 Funcionalidades

## 📡 Monitoramento de Dados

O sistema monitora:

- Temperatura da nave
- Energia da missão
- Comunicação da operação
- Nível de oxigênio
- Status dos módulos da nave

---

## ⚠ Geração Automática de Alertas

O sistema identifica automaticamente situações críticas como:

- Temperatura elevada
- Energia baixa
- Comunicação instável
- Oxigênio abaixo do ideal
- Falhas nos módulos operacionais

---

## 🤖 Tomada de Decisão Automática

Quando uma situação crítica é detectada, o sistema executa ações automáticas como:

- Ativação do sistema de resfriamento
- Modo econômico de energia
- Reinicialização da antena
- Liberação emergencial de oxigênio
- Acionamento da equipe técnica

---

## 🌱 Sustentabilidade Energética

O sistema também realiza análises relacionadas ao consumo energético da missão, aplicando estratégias sustentáveis para otimização da energia operacional.
